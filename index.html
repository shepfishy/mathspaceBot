Mathspace Bot

Automate your math homework with AI-powered solutions that extract problems from Mathspace and solve them step-by-step using Google's Gemini AI.

View on GitHub Download & Setup

Features

Automatic Problem Detection

Monitors Mathspace navigation and detects when you visit a problem page, no manual input required.

AI-Powered Solutions

Leverages Google's Gemini AI to generate detailed, step-by-step solutions to math problems.

Smart Extraction

Extracts both problem headers and subproblems from various Mathspace page structures.

Problem Saving

Automatically saves questions and solutions to text files for later review and reference.

Keyboard Shortcuts

Control the bot using convenient keyboard combinations for efficiency.

Fully Automated

Set it up once and the bot handles the rest, streamlining your workflow.

How It Works

1

Detection

The bot monitors your browser and automatically detects when you navigate to a Mathspace problem.

2

Extraction

Problem text, equations, and images are extracted from the Mathspace page.

3

AI Processing

The extracted problem is sent to Google's Gemini AI, which formulates a comprehensive solution.

4

Saving Results

The question and solution are saved to a text file for your reference and study purposes.

Get Started

Prerequisites

  • Python 3.12.2
  • Google Chrome browser
  • A Gemini API key (get one from Google AI Studio)
  • Administrator privileges (required for keyboard shortcuts)

Installation

1. Clone the repository or download the code:

git clone https://github.com/shepfishy/puppeteer-mathspace.git
cd puppeteer-mathspace

2. Install the required Python packages:

pip install -r requirements.txt

3. Run the bot:

# Linux
python webmath.py

# Windows
py webmath.py

Usage

Once running, navigate to any math problem on Mathspace. The bot will automatically extract the question, solve it, and save the results.

Keyboard Shortcuts

Shortcut Action
Ctrl+Alt+R Re-process the current page
Ctrl+Alt+Q Quit the application

Troubleshooting

Permission Denied Errors

Try running the script as administrator, especially if keyboard shortcuts aren't working.

API Key Issues

If you get API errors, check that your Gemini API key is valid and properly configured.

Element Not Found Errors

Mathspace's HTML structure may change. If extraction fails, try updating the CSS selectors in the code.

Chrome Driver Issues

If Chrome doesn't open, ensure you have the latest version of Chrome installed.
