**1. Understanding the Problem**

We need to find the distance between two points in a coordinate plane. The origin is the point (0, 0), and we are given the point P(11, 7). We can use the distance formula to solve this.

**2. The Distance Formula**

The distance *d* between two points (x₁, y₁) and (x₂, y₂) in a coordinate plane is given by:

d = √((x₂ - x₁)² + (y₂ - y₁)²)

*   Let (x₂, y₂) be the point P (11, 7).

Substitute these values into the distance formula:

d = √((11 - 0)² + (7 - 0)²)

d = √((11)² + (7)²)
d = √(121 + 49)
d = √(170)

d ≈ 13.0384
