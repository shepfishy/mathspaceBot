QUESTION:
How far is the given point P (15,8) from the origin?

SOLUTION:

**1. Understanding the Problem**

We need to find the distance between two points in a coordinate plane. We can use the distance formula for this. The distance formula is derived from the Pythagorean theorem.

**2. The Distance Formula**

The distance *d* between two points (x₁, y₁) and (x₂, y₂) is given by:

d = √((x₂ - x₁)² + (y₂ - y₁)² )

**3. Identifying the Coordinates**

*   Point P: (x₁, y₁) = (15, 8)
*   Origin: (x₂, y₂) = (0, 0)

**4. Plugging the Values into the Formula**

d = √((0 - 15)² + (0 - 8)²)

**5. Simplifying the Expression**

d = √((-15)² + (-8)²)
d = √(225 + 64)

**6. Calculating the Sum**

d = √(289)

**7. Finding the Square Root**

d = 17

**Answer:**

The distance between the point P(15, 8) and the origin is 17 units.
